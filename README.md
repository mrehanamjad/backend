# backend
backend in node express


## Project Setup:

express:
```bash
npm i express
```
nodemon: 
```bash
npm i nodemon
```
prettier
- install prettier as dev dependency
```bash
npm i -D prettier
```
- now make to files `.prettierrc` & `.prettierignore`
